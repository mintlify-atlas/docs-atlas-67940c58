---
title: "Eventos Urbanos API"
description: "API endpoints for managing urban events (eventos urbanos) in the Cinefinder application"
---

The Eventos Urbanos API allows you to create, read, update, and delete urban event records in the Cinefinder system.

## Get All Urban Events

<CodeGroup>
```bash cURL
curl -X GET http://localhost:8080/eventosUrbanos
```
</CodeGroup>

Retrieves a list of all urban events in the system.

### Response

<ResponseField name="[]" type="array">
  Array of urban event objects
  
  <Expandable title="Urban Event Object">
    <ResponseField name="id" type="integer">
      Unique identifier for the urban event
    </ResponseField>
    
    <ResponseField name="titulo" type="string">
      Title of the urban event
    </ResponseField>
    
    <ResponseField name="descripcion" type="string">
      Description of the urban event
    </ResponseField>
    
    <ResponseField name="fecha" type="string">
      Date of the event in ISO format
    </ResponseField>
    
    <ResponseField name="lugar" type="string">
      Location where the event takes place
    </ResponseField>
    
    <ResponseField name="categoria_id" type="integer">
      ID of the category this event belongs to
    </ResponseField>
  </Expandable>
</ResponseField>

### Response Example

```json
[
  {
    "id": 1,
    "titulo": "Festival de Jazz",
    "descripcion": "Festival de jazz al aire libre",
    "fecha": "2026-06-15T00:00:00.000+00:00",
    "lugar": "Parque Central",
    "categoria_id": 4
  }
]
```

## Create Urban Event

<CodeGroup>
```bash cURL
curl -X POST http://localhost:8080/eventosUrbanos \
  -H "Content-Type: application/json" \
  -d '{
    "titulo": "Concierto Rock",
    "descripcion": "Concierto de rock en vivo",
    "fecha": "2026-07-20T00:00:00.000+00:00",
    "lugar": "Estadio Municipal",
    "categoria_id": 5
  }'
```
</CodeGroup>

Creates a new urban event in the system.

### Request Body

<ParamField body="titulo" type="string" required>
  Title of the urban event
</ParamField>

<ParamField body="descripcion" type="string" required>
  Description of the urban event
</ParamField>

<ParamField body="fecha" type="string" required>
  Date of the event in ISO format (e.g., "2026-07-20T00:00:00.000+00:00")
</ParamField>

<ParamField body="lugar" type="string" required>
  Location where the event takes place
</ParamField>

<ParamField body="categoria_id" type="integer" required>
  ID of the category this event belongs to
</ParamField>

### Response

Returns the created urban event object with HTTP status 201 (Created).

<ResponseField name="id" type="integer">
  Unique identifier for the newly created urban event
</ResponseField>

<ResponseField name="titulo" type="string">
  Title of the urban event
</ResponseField>

<ResponseField name="descripcion" type="string">
  Description of the urban event
</ResponseField>

<ResponseField name="fecha" type="string">
  Date of the event in ISO format
</ResponseField>

<ResponseField name="lugar" type="string">
  Location where the event takes place
</ResponseField>

<ResponseField name="categoria_id" type="integer">
  ID of the category this event belongs to
</ResponseField>

### Response Example

```json
{
  "id": 2,
  "titulo": "Concierto Rock",
  "descripcion": "Concierto de rock en vivo",
  "fecha": "2026-07-20T00:00:00.000+00:00",
  "lugar": "Estadio Municipal",
  "categoria_id": 5
}
```

## Delete Urban Event

<CodeGroup>
```bash cURL
curl -X DELETE http://localhost:8080/eventosUrbanos/{id}
```
</CodeGroup>

Deletes an urban event from the system.

### Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the urban event to delete
</ParamField>

### Response

<ResponseField name="type" type="string">
  Response type, always "Success"
</ResponseField>

<ResponseField name="code" type="string">
  HTTP status code (200)
</ResponseField>

<ResponseField name="details" type="string">
  Success message with the deleted event ID
</ResponseField>

<ResponseField name="location" type="string">
  Request URI that was called
</ResponseField>

<ResponseField name="timestamp" type="string">
  Timestamp of the operation
</ResponseField>

### Response Example

```json
{
  "type": "Success",
  "code": "200",
  "details": "el evento con el id: 1. fue eliminado correctamente.",
  "location": "/eventosUrbanos/1",
  "timestamp": "2026-03-04T10:30:00"
}
```
